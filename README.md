# Explore Histórias

Um jogo de exploração 2D desenvolvido na **Godot Engine**. O objetivo do projeto é criar uma experiência imersiva
e simples, onde o jogador pode explorar um mundo rico em detalhes e interagir com os elementos ao seu redor.

## 🚀 Sobre o Projeto

**Explore Histórias** é um projeto autoral focado em mecânicas de interação e movimentação em ambiente 2D. O jogador
assume o controle de um personagem que deve percorrer diferentes cenários, descobrindo segredos e interagindo com
objetos ou NPCs para progredir na narrativa.

### Funcionalidades:
- [x] Movimentação 2D fluida.
- [x] Sistema de interação com o cenário.
- [x] Exploração de mapas em Pixel Art.

## 🛠️ Tecnologias Utilizadas

- **Engine:** [Godot Engine](https://godotengine.org/)
- **Linguagem:** GDScript
- **Sistema Operacional de Desenvolvimento:** Pop!_OS Linux
